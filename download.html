Verify what you run

Every release ships a SHA256SUMS file listing the exact hash of each binary, plus a signature over that file made with the project's release key. Two checks, in order:

1. Check the hash

sha256sum --check SHA256SUMS
# daemonseed-x86_64.AppImage: OK

If the line does not say OK, stop — the file is not what was published.

2. Check the signature

The signature over SHA256SUMS confirms it came from the project and not a middleman. The release public key and the exact verification command are published alongside the first signed release, on the releases repository.
